# Destination Icons

This folder contains icon images for the destination navigation slider.

## Required Icons

Place the following icon files in this directory:

- explore.webp (26x26px or 32x32px for retina)
- kerala.webp
- europe.webp
- nepal.webp
- singapore.webp
- chardham.webp
- rajasthan.webp
- ladakh.webp
- thailand.webp
- bali.webp
- sri-lanka.webp
- himachal.webp
- north-east.webp
- dubai.webp
- uttar-pradesh.webp
- malaysia.webp
- andaman.webp
- maldives.webp

## Image Specifications

- Format: WebP (preferred) or PNG
- Size: 26x26px @ 1x, 32x32px @ 2x for retina displays
- Style: Simple, iconic representations of each destination
- Background: Transparent or white

## Fallback

If icon images are not available, the component will display a broken image icon. You can:
1. Add actual icon images to this folder
2. Use emoji icons instead
3. Use an icon library like Lucide React

## Example Sources

You can get destination icons from:
- Custom design
- Icon libraries (Flaticon, Icons8, etc.)
- Emoji representations
- SVG icons
