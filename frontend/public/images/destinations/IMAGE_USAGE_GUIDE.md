## Integration Points

### 1. Package Cards
Each package card on the listings page shows **one main image** that represents the trip's essence.

### 2. Package Detail Gallery
Package detail pages show a **5-image gallery** with:
- 1 main hero image (largest)
- 4 supporting images (grid)

### 3. Itinerary Days
Each day in the itinerary can have **1-2 images** showing activities or destinations for that day.

### 4. Mobile Optimization
All images are in `.webp` format for optimal mobile loading and performance.

## Best Practices

### DO's
✅ Rotate images across packages to keep fresh
✅ Use season-appropriate images for dates
✅ Match image mood to package type
✅ Show 5 diverse images in galleries
✅ Use romantic images for couples packages
✅ Use adventure images for boys packages

### DON'Ts
❌ Don't use winter images for summer packages
❌ Don't repeat same image in adjacent packages
❌ Don't mix contradicting moods
❌ Don't use images that look AI-generated
❌ Don't use images that oversell
